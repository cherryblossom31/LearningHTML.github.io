Students are to play with h-tags, p-tag, a-tag, etc. in the viewport with inline style settings.
Steps in building an Image
  • Pick an Image (careful of the file extension in the URL)
  • Record teh URL, cite it, comment for the date accessed, and record the origonal dimensions
  • Note: opening in a new tab will eventually provide a http or https URL with a file extention
  • Use blockquote and comment as appropirate
  • Use img-tag with attributes src, alt, title, and style with width-height
  • Compare the same image with two sizes of the viewport (%25 & 100%)
